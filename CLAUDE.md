# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Type

This is an **educational content repository** containing Agile/Lean training materials for a Kanban/Scrum/Scrumban course, including Professional Scrum with Kanban (PSK) certification preparation.

**Note:** This is not a software project - there is no source code, build system, or tests.

## Content Structure

### Lecture Transcripts
- **Individual files:** `ID.Lesson Name.txt` format (e.g., `1.Introduction.txt`, `29.Lead Time.txt`)
- **Bundled files:** `39-51.txt`, `52-85.txt`, `86-88.txt`, `90-118.txt`, `119-125.txt` (grouped lectures)
- **Lesson delimiter:** Bundled files use `ID.` format (e.g., `119.`, `120.`) to separate lessons

### Course Organization (125 modules total)
| Lectures | Topic |
|----------|-------|
| 1-9 | Introduction & Kanban fundamentals |
| 10-25 | Kanban board building |
| 26-27 | WIP management |
| 28-38 | Metrics and CFD |
| 39-50 | Adoption framework |
| 51-75 | Scrum methodology |

| 99-125 | Leadership and advanced topics |

### Key Reference Files
- `CourseToC.txt` - Master index mapping `lesson_id` to lesson names (use this to find which .txt file contains a specific lesson)
- `ReadBeforeTakePSK1.txt` - PSK certification exam preparation guidance

### Supporting Materials (`others/` directory)
- **PDFs:** 19 reference documents (Kanban Guide, Scrum Guide, CFD guides, etc.)
- **Spreadsheets:** Templates for CFD, Lead Time, Throughput analysis, GTD worksheet

## Main Topics Covered

- Kanban Method and Kanban System
- Scrum framework and ceremonies
- Scrumban hybrid approach
- Work-in-Progress (WIP) limits
- Metrics: Lead Time, Cycle Time, Throughput, Cumulative Flow Diagrams
- Professional Scrum with Kanban (PSK) certification
- Leadership development and Getting Things Done (GTD)
